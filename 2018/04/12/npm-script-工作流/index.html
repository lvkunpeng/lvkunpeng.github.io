npm run 执行命令

使用npm run xxx 可以执行写在package.json文件中 “scirps” 里面的命令

在执行 npm run xxx 之前，npm会将node_modules/.bin添加到环境变量 $PATH的前面，也就是所有可执行文件的npm依赖都可以直接在 npm script 中直接调用，无需在npm script 中加入完整路径 eg:”./node_modules/.bin/xxxx **.js”
